# Salopets-Website

_Sitio web empresarial creado por Intelloidea para Veterinaria Salopets

### Pre-requisitos 📋
```
-PHP V7
-Servidor Web Apache
```
## Deployment 📦

_El deploy se debe realizar por medio de un cliente FTP_
```
-FTP
```
